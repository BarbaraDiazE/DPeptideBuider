User Guide



Numerate combinatorial peptide libraries

Fill out the form in home page


Step 1. Move the mouse over an "options" button, choose desired options from dropdown list

Linear: Select amino acid dataset

Methylated: Select amino acid with N-Methylation


Step 2. From dropdown list, select a single topology, or both of them.


Step 3. Select a number between 2 and 6 to set up the peptide length.


Step 4. click the “Submit” button to perform the enumaration


Libraries will appear in a new window, like the one at next screenshot.
